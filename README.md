# 수파자 프론트엔드 과제

안녕하세요! 프론트엔드 개발자 홍예찬입니다. 과제에 대한 소스코드가 어떻게 이루어져있는지 README 파일을 통해 설명해드리도록 하겠습니다.

## 기술 스택

- React(React Hooks)<br>
- SCSS<br>
- React-Router-Dom(로그인 성공 시, 페이지를 넘어가기 위해 사용했습니다.)<br>

## 컴포넌트 단위 분류
